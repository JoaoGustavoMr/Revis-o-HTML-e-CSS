O que é CSS?

Introdução ao CSS

CSS (Cascading Style Sheets) é uma linguagem usada para descrever a apresentação de documentos escritos em HTML ou XML. Ela define como os elementos devem ser exibidos em uma página web, permitindo a separação entre o conteúdo e o design visual, como cores, fontes e layout. CSS facilita a criação de layouts e estilos consistentes em todo o site, e possibilita a personalização visual sem alterar o conteúdo HTML.

Seletores

Seletores de Tag: Selecionam elementos HTML com base no nome da tag.

Exemplo: p aplica estilos a todos os elementos <p> do HTML.

p {
color: blue;
}

Seletores de Classe: Selecionam elementos com um atributo class específico.

Exemplo: .highlight aplica estilos a todos os elementos com class="highlight".

.highlight {
background-color: yellow;
}

Seletores de ID: Selecionam elementos HTML com base no nome da tag.

Exemplo: #header aplica estilos ao elemento com id="header".

#main-header {
font-size: 24px;
color: green;
}

Propriedades e Valores

Propriedades: Definem os estilos que você deseja aplicar (ex.: color, font-size, margin).

Valores: Especificam o valor para a propriedade (ex.: blue, 16px, 10px).

Classes e IDs

Classes: Usadas para aplicar o mesmo estilo a vários elementos. Iniciam com um ponto (.) e podem ser reutilizadas.

IDs: Usados para aplicar estilos a um único elemento. Iniciam com um # e devem ser únicos por página.
