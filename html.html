O que é HTML?

Introdução ao HTML

A Linguagem de Marcação de Hipertexto (HTML) é uma linguagem de computador que compõe a maior parte das páginas da internet e dos aplicativos online. Um hipertexto é um texto usado para fazer referência a outros textos, enquanto uma linguagem de marcação é composta por uma série de marcações que dizem para os servidores da web qual é o estilo e a estrutura de um documento.

O HTML não é considerado uma linguagem de programação, já que ele não pode criar funcionalidades dinâmicas. Ao invés disso, com o HTML, os usuários podem criar e estruturar seções, parágrafos e links usando elementos, tags e atributos.

Tags HTML Básicas

Páragrafo <p>

A tag <p> é usada para definir parágrafos de texto.

Exemplo de Parágrafo:

<p> O meu nome é João Gustavo </p>


Tags para Listas <li>, <ul>, <ol>

Existem dois tipos principais de listas: não ordenadas <ul> e ordenadas <ol>. Os itens das listas são definidos com a tag <li>.

Exemplo de lista não ordenada:

  • Futebol
  • Basquete
  • Vôlei

Exemplo de lista ordenada:

  1. São Paulo
  2. Flamengo
  3. Palmeiras

Tags de Tabela <table>, <tr>, <th>, <td>

<table>: A tag <table> é usada para criar uma tabela em HTML.
<tr>: A tag <tr> define uma linha em uma tabela.
<th>: A tag <th> define uma célula de cabeçalho em uma tabela.
<td>: A tag <td> define uma célula padrão em uma tabela.

Tags de Imagem <img>

A tag <img> apresenta na tela do navegador uma imagem, carregada a partir de um arquivo externo ao código HTML.


Tag para âncora de hyperlink <a>

O elemento <a> em HTML (ou elemento âncora), com o atributo href cria-se um hiperlink para páginas web, arquivos, endereços de emails, ligações na mesma página ou endereços na URL. O conteúdo dentro de cada <a> precisará indicar o destino do link.

Exemplo de Tag <a>:

Visite o site do Google

Nesse caso a frase "Visite o site do Google" é um hiperlink para essa página em destaque.
